The weather data is collected from [http://w2.weather.gov/](http://w2.weather.gov/).

Here are the columns:
* Date
* Maximum Temperature
* Minimum Temperature
* Average Temperature
* Temperature Departure
* HDD
* CDD
* Precipitation
* New Snow
* Snow Depth
